Lists

There are a few options to style lists in a custom way. For the most part these are default.

Unordered List

This is the standard markup for an unordered list. You still need to add the class .list to maintain consistency.

  • Item One
  • Item Two
  • Item Three
  • Item Four
                
                    <ul class="list">
                        <li>Item One</li>
                        <li>Item Two</li>
                        <li>Item Three</li>
                        <li>Item Four</li>
                    </ul>

Ordered List

This is the standard markup for an ordered list. You still need to add the class .list to maintain consistency.

  1. Item One
  2. Item Two
  3. Item Three
  4. Item Four
                
                    <ol class="list">
                        <li>Item One</li>
                        <li>Item Two</li>
                        <li>Item Three</li>
                        <li>Item Four</li>
                    </ol>

Stripped List

Add the class .list-stripped to remove any default styles to lists. This works with both list types ul, ol.

  1. Item One
  2. Item Two
  3. Item Three
  4. Item Four
                
                    <ol class="list list-stripped">
                        <li>Item One</li>
                        <li>Item Two</li>
                        <li>Item Three</li>
                        <li>Item Four</li>
                    </ol>

Tiny List

Add the class .list-tiny and the font size will be smaller. You can combine this with any of the above markup ul, ol or styles.

  1. Item One
  2. Item Two
  3. Item Three
  4. Item Four
                
                    <ol class="list list-stripped list-tiny">
                        <li>Item One</li>
                        <li>Item Two</li>
                        <li>Item Three</li>
                        <li>Item Four</li>
                    </ol>

Data List

Another kind of lists using different markup is the datalist. These are pretty basic styles, but you can customize if you like. They are located in the same scss file as the lists.

Item One
This is the description for Item One.
Item Two
This is the description for Item Two.
Item Three
This is the description for Item Three.
Item Four
This is the description for Item Four.
                
                    <dl class="list">
                        <dt>Item One</dt>
                        <dd>This is the description for Item One.</dd>
                        <dt>Item Two</dt>
                        <dd>This is the description for Item Two.</dd>
                        <dt>Item Three</dt>
                        <dd>This is the description for Item Three.</dd>
                        <dt>Item Four</dt>
                        <dd>This is the description for Item Four.</dd>
                    </dl>
