Checkbox

These should be contained within the .field container and use standard markup.

                
                    <div class="field">
                            <input id="checkbox-1" class="checkbox" type="checkbox" role="checkbox" aria-checked="false" value="1" name="checkbox-1" />
                            <label for="checkbox-1" class="checkbox-label">Checkbox label</label>
                    </div>
                     
                    <div class="field">
                            <input id="checkbox-2" class="checkbox" type="checkbox" role="checkbox" aria-checked="false" value="2" name="checkbox-2" />
                            <label for="checkbox-2" class="checkbox-label">A longer Checkbox label that spans two lines. This is to test if the checkbox is properly aligned to the top of this text. It shouldn't be aligned to the center.</label>
                    </div>

Auto Selecting a Checkbox Option

To automatically select an option, toggle the aria-checked="false" attribute to be aria-checked="true". This can be changed by the user if they unselect it.

                
                    <div class="field">
                            <input id="checkbox-3" class="checkbox" type="checkbox" role="checkbox" aria-checked="false" value="3" name="checkbox" />
                            <label for="checkbox-3" class="checkbox-label">Checkbox label</label>
                    </div>
                     
                    <div class="field">
                            <input id="checkbox-4" class="checkbox" type="checkbox" role="checkbox" aria-checked="true" value="4" name="checkbox" />
                            <label for="checkbox-4" class="checkbox-label">A longer Checkbox label that spans two lines. This is to test if the checkbox is properly aligned to the top of this text. It shouldn't be aligned to the center.</label>
                    </div>

Disabled State

To make an option unclickable simply add the disabled attribute to .checkbox input.

                
                    <div class="field">
                            <input id="checkbox-3" class="checkbox" type="checkbox" role="checkbox" aria-checked="false" value="3" name="checkbox" />
                            <label for="checkbox-3" class="checkbox-label">Checkbox label</label>
                    </div>
                     
                    <div class="field">
                            <input id="checkbox-4" class="checkbox" type="checkbox" role="checkbox" aria-checked="true" value="4" name="checkbox" />
                            <label for="checkbox-4" class="checkbox-label">Checkbox label (disabled)</label>
                    </div>

Accessibility & ARIA Roles

When updating a checkbox, we need to add the attributes for Accessibility. These attributes need to be updated using javascript. This code is located within the components.js file.
