  • Usage and How to show / remove loader

Spinner

You can add a spinner to show that something is loading. To use a smaller version, add the class .loader-small. There should only be one spinner showing on the page at a time. Too many spinners can be disorientating.

                    <div class="loader">
                        <span class="icon-spinner></span>
                    </div>
                
                    <div class="loader loader-small">
                        <span class="icon-spinner></span>
                    </div>

Inline Loader

For when space is limited or you want to load something inline, you can use this version. Again, try to keep the amount of loading indicators on a page to a minimum.

                    <div class="loader loader-inline">
                        <div class="bounce-1></div>
                        <div class="bounce-2></div>
                        <div class="bounce-3></div>
                    </div>
