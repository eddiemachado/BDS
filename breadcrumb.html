Breadcrumb

Used for page navigation, these are useful at the top of a page to show where you are.

All Links

This usage has each breadcrumb as a link.

Breadcrumb 1
Breadcrumb 2
Breadcrumb 3 
  					<nav class="breadcrumb" aria-label="breadcrumb">
    				        <div class="crumb">
	    	    			        <a href="#" class="crumb-link">Breadcrumb 1</a>
    					    </div>
        				    <div class="crumb">
        					        <a href="#" class="crumb-link">Breadcrumb 2</a>
    	    			    </div>
        				    <div class="crumb">
	        				        <a href="#" class="crumb-link">Breadcrumb 3</a>
	        			    </div>
	        		</nav>

No link on Last Item

Sometimes you don't need the last item to be a link because that's the page you're on. In that instance, you can just use a span instead of a link for the last item.

Breadcrumb 1
Breadcrumb 2
Breadcrumb 3 
  					<nav class="breadcrumb" aria-label="breadcrumb">
    				        <div class="crumb">
	    	    			        <a href="#" class="crumb-link">Breadcrumb 1</a>
    					    </div>
        				    <div class="crumb">
        					        <a href="#" class="crumb-link">Breadcrumb 2</a>
    	    			    </div>
        				    <div class="crumb">
	        				        <span class="crumb-current">Breadcrumb 3</span>
	        			    </div>
	        		</nav>
