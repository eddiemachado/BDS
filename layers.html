Layers

This is how to present content and hierarchy though depth. You can use these classes for things like sticky navs, modals, or tooltips. You can also use them as ways to elevate cards and content.

Adding styles to CSS

You can also use the CSS variables to apply these styles. They are already added to many components by default, but you can add these styles to any custom element if you choose.

Layer
Class
Sass Variable
Style
Flat
.layer-flat
$layFlat
box-shadow: none
Raised
.layer-raised
$layRaised
box-shadow: 0 1px 2px 0 rgba(0,0,0,0.10)
Overlay
.layer-overlay
$layOverlay
box-shadow: 0 4px 8px 0 rgba(0,0,0,0.10)
Sticky
.layer-sticky
$laySticky
box-shadow: 0 6px 12px 0 rgba(0,0,0,0.10)
Pop
.layer-pop
$layPop
box-shadow: 0 12px 24px 0 rgba(0,0,0,0.10)
