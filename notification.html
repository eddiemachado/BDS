Notifications

These are alert boxes used for things like error messages, success messages, or even warnings. Basically, anytime you want to grab attention within the UI for something that's happened.

Standard Notification

This notification doesn't have any specific style attached to it, so I wouldn't really use it. But hey, it's your site.

Standard Notification

You can put anything you want in this box, it should be formatted the same as it is everywhere else.

                
                    <div class="notification">
                          <h3>Standard Notification</h3>
                          <p>You can put anything you want in this box, it should be formatted the same as it is everywhere else.</p>
                    </div>

Danger Notification

By adding the class .noti-danger you can change the color of the notification to your danger support color.

Danger

This is a danger notification. You can put anything you want in here.

                
                    <div class="notification noti-danger">
                          <h3>Danger</h3>
                          <p>This is a danger notification. You can put anything you want in here.</p>
                    </div>
