Text Input

Text inputs can work with or without descriptions.

                
                    <div class="field">
                      <label for="text-1" class="field-label">Text Input Label</label>
                      <input id="text-1" type="text" class="input text-input" aria-required="false" aria-label="Placeholder Text" placeholder="Placeholder Text" />
                      <div class="input-feedback" data-feedback-field="text-1"></div>
                    </div>
                     

Input Types

There are several different input types that can be used in several different situations. There's probably more and there will be more in the future so keep an eye out when you're creating your forms.

              
                  <div class="field">
                    <label for="password-1" class="field-label">Password Input</label>
                    <input id="password-1" type="password" class="input text-password" aria-required="false" aria-label="Enter your Password" placeholder="Enter your Password" />
                    <span class="input-password-toggle"></span>
                    <div class="input-feedback" data-feedback-field="password-1"></div>
                  </div>
                   
                  <div class="field">
                    <label for="email-1" class="field-label">Email Input</label>
                    <input id="email-1" type="email" class="input input-email" aria-required="false" aria-label="Enter your Email" placeholder="username@domain.com" />
                    <div class="input-feedback" data-feedback-field="email-1"></div>
                  </div>
                   
                  <div class="field">
                    <label for="number-1" class="field-label">Number Input</label>
                    <input id="number-1" type="number" class="input input-number" aria-required="false" aria-label="Enter a number 0-24" placeholder="0-24" min="0" max="24" />
                    <div class="input-feedback" data-feedback-field="number-1"></div>
                  </div>
                   
                  <div class="field">
                    <label for="url-1" class="field-label">URL Input</label>
                    <input id="url-1" type="url" class="input input-url" aria-required="false" aria-label="Enter the URL" placeholder="http://www.example.com" />
                    <div class="input-feedback" data-feedback-field="url-1"></div>
                  </div>

Textarea

These should work the same as standard text inputs. You can adjust the default height by changing the rows and cols attributes.

            
                <div class="field">
                  <label for="textarea-1" class="field-label">Textarea Label</label>
                  <textarea id="textarea-1" class="input textarea" rows="3" cols="50" aria-required="false" aria-label="Placeholder Text" placeholder="Placeholder Text"></textarea>
                  <div class="input-feedback" data-feedback-field="textarea-1"></div>
                </div>

Input Descriptions

To add a description you need to add the .input-desc class to the input. You'll also need to put the descriptive text in the .input-feedback div. This text would be replaced with the validation text if there is validation tied to the field.

This is a field with a description
                
                    <div class="field">
                      <label for="text-2" class="field-label">Text Input Label</label>
                      <input id="text-2" type="text" class="input text-input input-desc" aria-required="false" aria-label="Placeholder Text" placeholder="Placeholder Text" />
                      <div class="input-feedback" data-feedback-field="text-2">This is a field with a description</div>
                    </div>

Validation

Each input should be validated, either on form submission or inline. These are the positive states. To add this state, add the valid to the input. You can have an optional message simply by adding text inside the .input-feedback div. Below you can see versions where text was added and one where nothing was added.

This is a success message
                
                    <div class="field">
                      <label for="text-3" class="field-label">Text Input Label</label>
                      <input id="text-3" type="text" class="input text-input" aria-required="false" aria-label="Placeholder Text" placeholder="Placeholder Text" valid />
                      <div class="input-feedback" data-feedback-field="text-3">This is a success message</div>
                    </div>
                     
                    <div class="field">
                      <label for="text-4" class="field-label">Text Input Label</label>
                      <input id="text-4" type="text" class="input text-input" aria-required="false" aria-label="Placeholder Text" placeholder="Placeholder Text" valid />
                      <div class="input-feedback" data-feedback-field="text-4"></div>
                    </div>

Invalid Fields

Negative states work the same, you can simply add the invalid to the input. The feedback messages work the same as the valid states. I reccomend using the descriptions to help users understand what went wrong.

This is a success message
                
                    <div class="field">
                      <label for="text-5" class="field-label">Text Input Label</label>
                      <input id="text-5" type="text" class="input text-input" aria-required="false" aria-label="Placeholder Text" placeholder="Placeholder Text" invalid />
                      <div class="input-feedback" data-feedback-field="text-5">This is an error message</div>
                    </div>

Disabled Fields

By adding the disabled field to the input, the state will be uneditable. You can also add a descriptive message if you like by using the same methods as above.

                
                    <div class="field">
                      <label for="text-6" class="field-label">Text Input Label</label>
                      <input id="text-6" type="text" class="input text-input" aria-required="false" aria-label="Placeholder Text" placeholder="Placeholder Text" disabled />
                      <div class="input-feedback" data-feedback-field="text-6"></div>
                    </div>

Adding Icons to Inputs

You can add an icon to any text input by adding the icon class to the .field-label. Below are some examples, as well as some with validation active.

This is a valid url
                
                    <div class="field">
                      <label for="text-1" class="field-label icon-github">Github Account URL</label>
                      <input id="url-2" type="text" class="input input-url" aria-required="false" aria-label="Enter your Github URL" placeholder="http://github.com/username" />
                      <div class="input-feedback" data-feedback-field="url-2"></div>
                    </div>
                     
                    <div class="field">
                      <label for="url-3" class="field-label icon-twitch">Twitch Account URL</label>
                      <input id="url-3" type="text" class="input input-url input-desc" aria-required="false" aria-label="Enter your Twitch URL" placeholder="http://twitch.tv/username" value="http://twitch.tv/somedumbguy" valid />
                      <div class="input-feedback" data-feedback-field="url-3">This is a valid url</div>
                    </div>
