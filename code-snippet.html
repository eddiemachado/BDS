Code Snippet

A simple way to display code on a page. This is usually handled by a library or framework as it can be a bit tedious to setup. Eitherway here are some default styles.

Basic & Dark Options

Either one works, just be consistent in their use. To use the dark version, simple add the class .code-snippet-dark.

                
                    <pre class="code-snippet">
                            <code class="line-numbers>
                                    ...code here...
                            </code>
                    </pre>
                
              
                
                    <pre class="code-snippet code-snippet-dark">
                            <code class="line-numbers>
                                    ...code here...
                            </code>
                    </pre>

Formatting Options

When adding code within the pre element, you'll need to use HTML entities in order for them to be displayed properly. To do this, you'll want to familiarize yourelf with HTML Entities. There are also some ways to highlight things within a code snippet. To use these you do not need to use entity code around what you're highlighting.

                
                  Using <em> around something in a code snippet will highlight it.
                
              
                
                  Using <strong> around something in a code snippet will make it bold.

Inline Code Snippets

You can also use inline options to highlight code within copy, as you've seen throughout this page.

The element .example is a good way to show off this feature.

                
                  <p>The element <span class="code-inline">.example</span> is a good way to show off this feature.</p>
