Basic & Dark Options

Either one works, just be consistent in their use. To use the dark version, simple add the class .code-snippet-dark.

                
                    <pre class="code-snippet">
                            <code class="line-numbers>
                                    ...code here...
                            </code>
                    </pre>
                
            
                
                    <pre class="code-snippet code-snippet-dark">
                            <code class="line-numbers>
                                    ...code here...
                            </code>
                    </pre>

Formatting Options

When adding code within the pre element, you'll need to use HTML entities in order for them to be displayed properly.
