Radio Options

Be sure to put radio buttons into a group. This way you can be sure they will function consistently.

Radio Group Label

                
                    <div class="field">
                          <h4 class="field-group-label">Radio Group Label</h4>
                          <ul class="radio-group list-stripped" role="radiogroup" aria-labelledby="field-group-label">
                                <li>
                                      <input id="radio-1" class="radio" type="radio" value="1" role="radio" aria-checked="false" name="radio" />
                                      <label for="radio-1" class="radio-label">Option 1</label>
                                </li>
                                <li>
                                      <input id="radio-2" class="radio" type="radio" value="2" role="radio" aria-checked="false" name="radio" />

                                </li>

                                </li>
                          </ul>
                    </div>

Auto Selecting a Radio Option

To automatically select an option, toggle the aria-checked="false" attribute to be aria-checked="true". This can be changed by the user if they select another option.

Disabled State

To make an option unclickable simply add the disabled attribute to .radio input.

Radio Group Label

                
                    <div class="field">
                          <h4 class="field-group-label">Radio Group Label</h4>
                          <ul class="radio-group list-stripped" role="radiogroup" aria-labelledby="field-group-label">
                                <li>
                                      <input id="radio-8" class="radio" type="radio" value="8" role="radio" aria-checked="false" name="radio" />
                                      <label for="radio-8" class="radio-label">Option 1</label>
                                </li>
                                <li>

                                </li>
                          </ul>
                    </div>
