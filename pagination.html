Pagination

Used for parts of the site that have multiple pages, like Search Results, you can use this to allow users to hit a specific page.

Things left to do

Basic Pagination

This works for most instances. Be aware of the ARIA attributes here as they will help us identify the current page. By using the attribute aria-current="true" we can highlight the page the user is one while also maintaining Accessibility compliance.

You can also shorten the pagination if there are too many pages or even use Previous / Next page options if the situation calls for it.
