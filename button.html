Button

There are a few different button types. By adding different classes to them, they will take on the different styles listed below.

Things left to do

Solid Button

There are a few different styles of buttons based on our colors.

Primary Buttons
These are used for the main actions on the page.
Secondary Buttons
These can be used when you can't use the primary buttons
Dark Buttons
This is the dark version
Light Buttons
This is the light version
Desctructive Buttons
These should be reserved for actions like deleting your account or something that is permanent.
                    <button type="button" class="btn btn-primary">Primary Button</button>
                    <button type="button" class="btn btn-secondary">Secondary Button</button>
                    <button type="button" class="btn btn-dark">Dark Button</button>
                    <button type="button" class="btn btn-light">Light Button</button>
                    <button type="button" class="btn btn-success">Success Button</button>
                    <button type="button" class="btn btn-destroy">Destructive Button</button>

Outline Button

Add the class .btn-outline to a button to use this version. Be mindful of the background color so there aren't contrast issues.

                    <button type="button" class="btn btn-outline btn-primary">Primary Button</button>
                    <button type="button" class="btn btn-outline btn-secondary">Secondary Button</button>
                    <button type="button" class="btn btn-outline btn-dark">Dark Button</button>
                    <button type="button" class="btn btn-outline btn-light">Light Button</button>
                    <button type="button" class="btn btn-outline btn-destroy">Destructive Button</button>

Transparent Button

To use the transparent version of the buttons, add the class .btn-outline to the button. Be mindful of where you use these icons and the contrast.

                    <button type="button" class="btn btn-transparent btn-primary">Primary Button</button>
                    <button type="button" class="btn btn-transparent btn-secondary">Secondary Button</button>
                    <button type="button" class="btn btn-transparent btn-dark">Dark Button</button>
                    <button type="button" class="btn btn-transparent btn-light">Light Button</button>
                    <button type="button" class="btn btn-transparent btn-destroy">Destructive Button</button>

Icon Button

In order to add an icon to a button, you'll need to add the class .btn-icon to the button as well as the class for the icon. You can find these classes in the Icon section of these guidelines.

                    <button type="button" class="btn btn-icon btn-primary icon-flash_on">Primary Icon Button</button>
                    <button type="button" class="btn btn-icon btn-outline btn-secondary icon-question_answer">Secondary Icon Button</button>
                    <button type="button" class="btn btn-icon btn-transparent btn-dark icon-star">Dark Icon Button</button>

Large Button

Add classes .btn-large or .btn-small for additional sizes.

                    <button type="button" class="btn btn-large btn-primary">Large Primary Button</button>
                    <button type="button" class="btn btn-large btn-icon btn-outline btn-secondary icon-speech">Large Secondary Icon Button</button>
                    <button type="button" class="btn btn-large btn-transparent btn-dark">Large Transparent Button</button>
                
                    <button type="button" class="btn btn-small btn-primary">Small Primary Button</button>
                    <button type="button" class="btn btn-small btn-icon btn-outline btn-secondary icon-bell">Small Secondary Icon Button</button>
                    <button type="button" class="btn btn-small btn-transparent btn-dark">Small Transparent Button</button>

Disabled State

Add the disabled attribute to create a button with a disabled state. All buttons with a disabled state will be styled the same as this state overrides all other styles.

                    <button type="button" class="btn btn-primary" disabled>Disabled Primary Button</button>
